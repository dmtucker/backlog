CLI¶

backlog¶

Manage a Backlog.

backlog [OPTIONS] COMMAND [ARGS]...

Options

--path <path>¶

Specify the path to use for the backlog file.

--version¶

Show the version and exit.

add¶

Add an entry to the backlog.

backlog add [OPTIONS] TITLE

Options

--note <note>¶

Add a note.

--priority <priority>¶

Specify a priority level.

Arguments

TITLE¶

Required argument

random¶

Select a random entry from the backlog.

backlog random [OPTIONS]

remove¶

Remove entries from the backlog.

backlog remove [OPTIONS] PATTERN

Options

--ask, --dont-ask¶

Prompt before deleting.

Arguments

PATTERN¶

Required argument

show¶

Show entries in the backlog.

backlog show [OPTIONS]

Options

--pattern <pattern>¶

Specify a search pattern for entry titles.
